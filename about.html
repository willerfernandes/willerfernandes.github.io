O Porquê

Esse site foi construido por mim com o intuito de demonstrar conhecimento sobre o desenvolvimento web e algumas de suas tecnologias mais tradicionais. Além disso, o portifólio online é uma forma prática, diferenciada e agradável de mostrar um pouco do meu trabalho.







Como

Página Inicial

Esse site foi construido utilizando puramente HTML + JavaScript + CSS. Os componentes utilizados foram interamente desenvovidos ou adotados do Boostrap e alterados para que atendam às necessidades específicas da página.

As páginas foram desenvolvidas de forma totalmente amigáveis à dispositivos móveis (mobile-friendly) e design dos componentes é responsivo e agradável para ser utilizado em telas menores.

A página inicial conta com um cabeçalho fixo (presente em todas a páginas) com a sua estrutura importada do Bootstrap e adaptada. Em seguida, um componente de apresentação do perfil, que contem a imagem de fundo junto à imagem de perfil centralizada. As seções seguintes são desenhadas em estrutura de grid para deixa o layout da página mais agradável. A seção de Qualificações técnicas conta com uma estrutura de Abas e, em cada uma delas, uma lista de "barras de progresso" com o nível de experiência em cada um dos tópicos.

A seção dos destaques técnicos mostra as principais experiências em imagens responsivas que são proporcionais ao tamanho da tela do dispositivo.

O "carousel" logo abaixo exibe de forma resumida algumas aplicações/sistemas já desenvovidos e as tecnologias envolvidas no processo. O JavaScript utilizado foi importado do bootstrap e utilizado puramento como foi desenvolvido.

Currículo Texto

O Objetivo dessa página é exibir uma versão de currículo tradicional, mais sóbrea, que pode ser impressa e utilizada para entrevistas de emprego, análise de currículos por setor de RH, etc. A página é um HMTL puro, com os botões de voltar (para a página inicial) e um botão de imprimir que dispara uma função no JavaScript que seleciona os elementos que serão impressos e solicita a impressão pelo navegador.

Contato

A página de contato mostra uma integração do site com uma API externa para fluxo de emails. Após preencher os dados, a mensagem é enviada para um servidor do Mail Chimp e pode ser visualizada e retornada. (O objetivo principal é mostrar a integração com a API. A caixa de emails do MailChimp não é monitorada. Caso tenha interesse em entrar em contato, favor utilizar o email no currículo).
